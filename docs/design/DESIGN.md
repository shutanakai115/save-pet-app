# デザインシステム: つもり貯金 (Tsumori Chokin)

## 1. ビジュアルテーマ & 雰囲気

**豚の貯金箱（Piggy Bank）をモチーフにした、日本語ポップなモバイル貯金アプリ。**

全体のトーンは明るく、励ましの雰囲気。アプリが「貯金を楽しい体験」に変えてくれる友達のような存在であること。

シンプルでかわいい豚キャラクターが UI のアクセントとして登場し、貯金額が増えるにつれて豚が「リッチになっていく」演出が加わる。

**雰囲気のキーワード:**

- **暖かく、親しみやすい** — Honey Gold と Coral Pink が主役。居心地の良い、前向きな印象
- **丸くて柔らかい** — すべての要素に丸角を採用。豚の丸いシルエットとリンク
- **励まし、お祝い** — 成功状態は目立ち、嬉しい演出。貯金が「達成感」に感じられること
- **すっきり、集中** — 遊び心がありつつも画面はクリーン。メインアクション（貯金記録）が常に前面に
- **日本語ポップ** — フレンドリーなタイポグラフィ、温かい色のコントラスト、キャラクター駆動のパーソナリティ

## 2. カラーパレット & 役割

### Primary: Warm Honey Gold

上品なウォームゴールド。コインや繁栄を連想させつつ、派手すぎず嫌味のない色。「温かいトーストに垂れるはちみつ」のような黄金色。

- **用途:** CTAボタン、プログレスバー、貯金額表示、達成バッジ、アクティブ状態
- **性格:** 前向き、報われる感覚、豊かさ

### Secondary: Soft Coral Pink

豚キャラクターから着想を得た、柔らかく温かいピンク。優しく近づきやすい色。

- **用途:** アクセント、キャラクターイラスト、ハイライト、カード背景、セカンダリボタン、カテゴリタグ
- **性格:** フレンドリー、かわいい、温かみ

### Success: やわらかいリーフグリーン

自然で柔らかいグリーン。貯金のマイルストーン達成や成功確認に使用。

- **用途:** 貯金成功フィードバック、目標達成表示
- **性格:** 新鮮、安心感、お祝い

### Error: 穏やかなウォームレッド

抑えめの暖色系レッド。エラーを伝えつつ、過度な不安を与えない色。

- **用途:** 削除アクション、バリデーションエラー
- **性格:** 明確だが攻撃的でない

### Neutral: ウォームグレー

微かな暖色の下地を持つグレー。冷たい印象や無機質さを避ける。

- **用途:** 本文テキスト、ボーダー、区切り線、無効状態、補助テキスト
- **性格:** 読みやすい、落ち着いた、控えめ

### Background: クリーミーホワイト → ピンク グラデーション

ほんのり暖色がかったオフホワイトからソフトピンクへの縦グラデーション。全画面で統一して使用。

- **用途:** ページ背景（全画面共通）
- **性格:** 柔らかい、温かい、ポップ
- **方向:** 上（クリーミーホワイト）→ 下（ソフトピンク）
- 装飾要素として、薄いピンクや黄色の丸いバブル/ドットを背景にあしらう

## 3. タイポグラフィ

### フォントファミリー

- **日本語:** Noto Sans JP — 丸みがあり親しみやすい書体。本文、見出し、UIラベルすべてに使用
- **欧文 / 数字:** Inter — クリーンでモダン、金額表示の視認性が高い

### ウェイトの使い分け

- **見出し:** Bold (700) — 明確な階層、確信のある印象
- **貯金額表示:** Bold (700) or Semibold (600) — 大きく目立つ、画面の「主役」
- **本文:** Regular (400) or Medium (500) — 快適な読み心地
- **キャプション / 補助:** Regular (400) — 軽い存在感、サポート的

### サイズ階層

- **ヒーロー貯金額:** 特大 — 画面で最大のテキスト、注目を集める
- **セクション見出し:** 大 — 明確なセクション区切り
- **本文 / リストアイテム:** 基本サイズ — モバイルで快適に読める（16px相当以上）
- **キャプション / タイムスタンプ:** 小 — 補助情報、ただし視認性は確保

## 4. コンポーネントスタイル

### ボタン

- **形状:** Pill-shaped（完全な丸角）— 柔らかくタップしやすい、豚の丸みとリンク
- **Primary CTA:** Honey Gold 背景 + 白またはダークテキスト、奥行きを出すための控えめなシャドウ
- **Secondary:** Coral Pink 背景、またはピンクボーダーのアウトライン
- **タッチターゲット:** 最低 44px（iOS推奨）、推奨 48px
- **フィードバック:** プレス時に軽いスケールダウン、微かな色の深まり

### カード

- **形状:** 大きめの丸角（16px以上のradius）
- **背景:** 白、またはごく薄いピンクのティント
- **シャドウ:** ウィスパーソフト — 繊細な奥行き、重さは感じさせない
- **スペーシング:** 十分な内側パディング、明確なコンテンツ階層

### 入力フォーム

- **形状:** 丸角（12px以上のradius）
- **ボーダー:** ライトウォームグレーのストローク、フォーカス時は Honey Gold
- **タッチターゲット:** 大きめ — モバイルでの親指入力に最適化
- **金額入力:** 特大フォント、目立ちやすく読みやすい

### ボトムシートモーダル（クイック入力）

- **表示:** 下からスライドアップ、柔らかいスプリングアニメーション
- **形状:** 上角のみ大きく丸角、下端は画面端と揃える
- **背景オーバーレイ:** 半透明のウォームトーン
- **ステップインジケーター:** 小さなドットまたはプログレスバー（1/3, 2/3, 3/3）
- **閉じる:** スワイプダウン、またはオーバーレイタップ

### ボトムナビゲーション

- **構成:** クリーン、ミニマル — 4タブ（ホーム / 履歴 / 目標 / 設定）
- **アクティブ:** Honey Gold のアイコン + ラベル
- **非アクティブ:** ウォームグレーのアイコン + ラベル
- **Safe Area:** ノッチ対応デバイス向けの適切な下部パディング

### 豚キャラクター

- **スタイル:** シンプル、ミニマルなイラスト — 線画またはフラット、最大2-3色
- **登場場所:** ホーム画面（メイン）、貯金完了のお祝い（Step 3）、履歴サマリー、空状態
- **パーソナリティ:** 明るく、進化する — 貯金が増えるとアクセサリーや表情が変わる（将来機能）

## 5. レイアウト原則

### グリッドシステム

- **基本単位:** 8px グリッド
- **半単位（4px）:** 微調整に許容（アイコンパディング、テキスト配置）

### モバイルファースト

- **基準ビューポート:** 幅 375px（iPhone SE / 標準モバイル）
- **コンテンツ幅:** モバイルではフルブリード、不要なサイドマージンなし
- **ページパディング:** 水平 16px（左右）

### スペーシング戦略

- セクション間は**余裕のあるホワイトスペース** — 軽やかで親しみやすい印象に
- 関連要素内は**タイトなグルーピング** — 明確な視覚的関係性
- **セクション間:** 24-32px
- **カード間:** 12-16px

### Safe Area

- **上部:** ステータスバーのクリアランス
- **下部:** ナビゲーションバー + ホームインジケーターパディング（ノッチデバイスで34px）

### スクロール挙動

- **ホーム画面:** 固定ボトムナビ付きの単一スクロールページ
- **履歴:** 月ヘッダーをスティッキー表示した無限スクロールまたはページネーション
- **目標:** 固定レイアウト（スクロール不要を想定）
- **モーダル:** コンテンツ溢れ時はボトムシート内でスクロール
